# Python Security Tools

A collection of Python-based cybersecurity tools, including encryption, file integrity monitoring, penetration testing, and web vulnerability scanning.

## Projects

### 1. File Encryption Tool
- AES-256 encryption/decryption for files
- GUI interface
- Usage: `python main.py`

### 2. File Integrity Monitor
- Monitors files for changes using hash verification
- Usage: `python file_integrity_monitor.py`

### 3. PenTest Toolkit
- Multi-threaded port scanner
- SSH brute-forcer
- Web vulnerability scanner (SQLi, XSS)
- Usage: `python main.py`

### 4. Web Vulnerability Scanner
- Scans web pages for common vulnerabilities
- Usage: `python scanner.py`
